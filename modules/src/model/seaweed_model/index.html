.construct_df_from_sections_for_date

source

.construct_df_from_sections_for_date(
   months
)

Constructs a dataframe from the data in the model for a given date. This uses the months since the beginning of the nuclear war. Mininum is -3, as the data starts before the war. Maximum is 357, as the data ends after the war.

Arguments

  • months : the months since the beginning of the nuclear war

Returns

a dataframe for the values at the given month
